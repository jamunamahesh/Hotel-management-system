# Hotel Management System using Python

This is a simple Hotel Management System project developed in Python.

## Features
- Show available rooms
- Book a room
- Check-In
- Check-Out
- Generate bill
- Exit the system

## Room Details
- Room 101 - Single - Rs.1000
- Room 102 - Double - Rs.1500
- Room 103 - Deluxe - Rs.2000

## Concepts Used
- Python Lists
- Nested Lists
- Loops
- Conditional Statements
- Input and Output

## How It Works
1. Show room details
2. Book a room if available
3. Check-in after booking
4. Check-out after stay
5. Bill is calculated based on room price and number of days
